To-Do List App

Aplikasi manajemen tugas harian berbasis ReactJS dengan komponen dinamis, state management, dan filter status — dibuat untuk memperdalam pemahaman React hooks dan interaksi antarkomponen.

✅

Latar Belakang

Saya ingin memperdalam kemampuan frontend, khususnya React. Membangun aplikasi To-Do List adalah cara yang efektif untuk berlatih mengelola state, props, dan interaksi antarkomponen pada satu aplikasi nyata yang interaktif.

Solusi

Saya mengembangkan aplikasi berbasis ReactJS untuk mencatat dan mengelola daftar tugas harian. Pengguna dapat menambah, menandai selesai, dan menghapus tugas, serta memfilter tugas berdasarkan statusnya. Styling dibuat dengan CSS sederhana agar tampilan tetap bersih dan responsif.

Fitur Utama

  • ▹Tambah, tandai selesai, dan hapus tugas.
  • ▹Filter tugas berdasarkan status: All, Active, Completed.
  • ▹Komponen dinamis dengan state management dasar.
  • ▹Antarmuka responsif dan mudah digunakan.

Peran Saya

Saya membangun aplikasi ini secara mandiri, menggunakan state dan props untuk mengatur alur data antarkomponen. Proyek ini memperdalam pemahaman saya tentang React hooks, pengelolaan data, dan desain antarmuka yang interaktif — menjadi bagian dari portofolio frontend saya.
